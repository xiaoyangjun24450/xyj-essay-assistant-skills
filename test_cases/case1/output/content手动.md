1. 

课题的背景和意义： 

本课题源于低轨卫星互联网在我国新型基础设施建设中的重要地位日益凸显。目前，传统地面网络在覆盖范围、传输时延和应急通信保障等方面存在局限，而低轨卫星互联网凭借广域覆盖、低时延传输和高稳定性的优势，已成为构建空天地一体化网络的关键支撑

[1]

。2020年4月，国家发改委首次将卫星互联网纳入“新基建”范畴

[2]

，卫星互联网建设已上升为国家战略性工程。低轨卫星互联网技术不仅能够实现全球范围内的网络覆盖，提供高速、低时延的互联网服务

[3]

，还能在地面网络因突发事件受损时发挥应急通信保障作用

[4]

。因此，低轨卫星互联网的建设不仅顺应全球通信技术的发展趋势，也契合国家基础设施完善与社会高质量发展的现实需求。 

编码调制技术作为低轨卫星互联网组网的核心技术，在当前空间网络研究领域备受关注。然而，低轨卫星互联网在高动态环境下面临信号衰减、多普勒频移等挑战

[5]

，传统短距离传输技术难以满足其对高鲁棒性的需求。作为一种兼具高度的可靠性和稳定性的技术，线性扩频（

Chirp Spread Spectrum, CSS

）调制具备远距离和抗干扰等优势

[6]

。现有研究表明，CSS调制对卫星互联网应用展现出良好的技术适应性

[7]

。此外，编码技术能够在有限带宽和低信噪比条件下实现低误码率传输，但现有编码技术在纠错性能与复杂度之间难以兼顾，无法同时满足低轨卫星通信对通信质量和低功耗的双重需求

[8]

。因此，研究卷积编码技术将为低轨卫星互联网建设提供有力支持，进一步完善国家新型基础设施建设。 

随着低轨卫星互联网成为全球通信网络升级的重要方向，相关编码调制技术已成为学术界与产业界的研究热点

[9][10][11]

。本课题旨在研究低轨卫星互联网的编码调制关键技术，以提升低轨卫星互联网通信的数据传输效率和系统性能。通过利用高效的调制技术和信道编码技术，解决低轨卫星通信中的信号衰减和干扰问题，进而提高卫星网络的传输鲁棒性与可靠性。具体而言，本研究将利用卷积编码与

CSS

调制技术的优势，提出一种创新性的编码调制方案，并通过仿真分析与实际测试评估所提方案在不同信道条件下的性能表现，验证其在提升数据传输效率和降低误码率方面的有效性。研究成果将为卫星通信系统的实际应用和优化提供理论支持，并对我国空天地一体化网络建设具有重要的工程应用价值。 

2. 

原始数据

(1)

加性高斯白噪声（

Additive White Gaussian Noise, AWGN

）建模数据。在本研究中，为简化分析过程，将实际环境中采集到的背景噪声和干扰信号建模为

AWGN

。该建模数据主要用于模拟通信系统在实际噪声和干扰环境下的工作状态，从而量化通信系统的抗噪声与抗干扰性能，评估系统可靠性。 

(2) 

二进制相移键控（

Binary Phase Shift Keying, BPSK

）调制仿真代码。基于

MATLAB

平台，原始信号经过

BPSK

调制后，通过

AWGN

和瑞利衰落信道传输并正确解调，计算误比特率（

Bit Error Rate, BER

）与误符号率（

Symbol Error Rate, SER

）数据，并绘制曲线图与理论值进行比较的仿真代码。 

(3) CSS

调制仿真代码。基于

MATLAB

平台，原始信号经过

CSS

调制后，通过

AWGN

和瑞利衰落信道传输，进行相干解调或非相干解调，计算BER数据并绘制

BER

曲线图的仿真代码。

5. 

毕业设计的主要内容 

(1)

面向低轨卫星互联网，研究现有调制和编码的基本理论和技术特点 

首先，低轨卫星互联网作为下一代通信技术的重要组成部分，凭借其广覆盖、低延时和高稳定的特性，已成为现代通信系统部署不可或缺的一环。由于低轨卫星具有较为复杂的动态信道特性，现有的调制技术难以满足其高鲁棒性的需求。因此，本研究将通过利用编码与调制技术各自优势，为低轨卫星互联网性能的提升提供新的思路和方法。 

其次，本研究将针对现有的调制技术（如

BPSK

、

OFDM

技术和

CSS

技术等）以及主流编码技术（如

LDPC

码、

Turbo

码、

Polar

码和卷积码等）展开深入分析，梳理各技术的基本理论，全面探讨其技术特点与适用场景，并结合各技术的优势与局限性，筛选并设计适用于本课题研究的最优编码调制方案，从而为制定课题技术路线提供坚实的理论依据。图1中展示了卷积码编码器的基本原理图。 

(2)

研究

CSS

调制技术、卷积编码的基本原理，搭建编码调制仿真测试平台 首先，传统调制技术在应对低轨卫星特有的快速信道变化和低信噪比环境时，面临性能瓶颈。尽管常见的编码技术如

LDPC

码、

Turbo

码和

Polar

码在提高数据传输可靠性方面表现出色，但其计算复杂度较高，难以在资源受限的低轨卫星网络中得到有效应用。而

CSS

技术具有低灵敏度接收和抗多普勒频偏的优越性，卷积编码具备实现简便和低复杂度的特点。故本研究选择利用

CSS

技术与卷积编码技术各自优势，以满足低轨卫星互联网在可靠性、稳定性和实现难度方面的需求。 

其次，在仿真平台的搭建方面，分别搭建

CSS

调制和卷积编码的仿真平台，并在各自的测试环境中进行初步性能测试。随后，进行

CSS

调制与卷积编码仿真平台的合并，构建一个完整的编码调制仿真系统，对其在低轨卫星环境下的性能进行了系统测试与分析。这一初步分析结果将为后续研究提供重要的数据支持。 

(3)

研究面向低轨卫星互联网的卷积编码

CSS

调制技术，分析其性能指标 

首先，在低轨卫星通信系统中，由于信号传播过程中受到多普勒效应、快速信道衰落以及低信噪比等因素的影响，信号衰落显著且复杂。深入研究低轨卫星信道的特性并对其进行精确建模，是开展相关研究的关键前提。本研究将基于低轨卫星信道特性建立信道模型，在此基础上进行面向低轨卫星互联网的卷积编码

CSS

调制方案的设计。 

其次，本研究将对所设计的方案在低轨卫星信道环境下的误码率性能展开深入分析，并推导出误码率的理论表达式。通过一系列仿真测试，评估所设计的编码调制方案在不同信道条件及参数配置下的性能表现，量化各因素对方案可靠性与效率的影响。基于仿真结果，进一步优化调制与编码方案的设计，旨在提升低轨卫星互联网通信的稳定性和传输效率，为实际应用提供理论支持与技术参考。
